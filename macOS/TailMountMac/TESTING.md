# macOS 实机验收清单

## 构建与启动

- [ ] `swift build -c release` 无错误。
- [ ] `scripts/build-app.sh` 生成 `dist/TailMount.app`。
- [ ] 应用图标、中文界面、浅色和深色模式显示正常。
- [ ] Apple Silicon 和 Intel/通用构建至少验证目标架构之一。

## 配置与安全

- [ ] 新建、切换、保存、删除多个配置正常。
- [ ] 重启后配置恢复，JSON 中不含密码。
- [ ] 勾选记住密码后 Keychain 中存在条目；取消后条目删除。
- [ ] 密钥路径支持 `~`，错误路径能给出清晰提示。

## 网络与挂载

- [ ] MagicDNS 名称和 100.x Tailscale IP 均能测试。
- [ ] Tailscale CLI 可用时显示 TSMP 延时；不可用时 TCP 回退正常。
- [ ] 密钥、密码两种方式都能挂载。
- [ ] Finder 可浏览、读取、写入、重命名和删除测试文件。
- [ ] 正常卸载、强制卸载和重新挂载正常。
- [ ] 断开 Wi-Fi、停止 Tailscale、服务器关机时不会卡死界面。
- [ ] 高延时、连续失败和恢复连接时状态及通知正确。

## 菜单栏与生命周期

- [ ] 关闭主窗口后进程继续运行，Dock 图标隐藏，菜单栏图标保留。
- [ ] 单击菜单“打开 TailMount”能恢复窗口。
- [ ] 菜单可打开当前磁盘和卸载。
- [ ] 只有“退出 TailMount”才结束进程。
- [ ] 登录时启动开关正常。
- [ ] 开启恢复挂载后，重新登录能恢复选中配置。
