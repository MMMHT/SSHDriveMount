# 更新记录

## 0.3.0 - 2026-07-14

- 关闭主窗口时最小化到 Windows 系统托盘，网络与挂载健康监测继续在后台运行。
- 增加托盘菜单，可恢复主界面、打开当前远程磁盘或明确退出程序。
- 再次从开始菜单启动已在后台运行的 TailMount 时，自动唤回现有窗口。
- 增加首次进入托盘时的后台运行提示，避免误以为程序已经退出。

## 0.2.0 - 2026-07-13

- 完成多服务器配置、SSH 测试、盘符挂载和卸载流程。
- 修复 SSHFS-Win 映射错误 1204。
- 修复 Windows PowerShell 5 导入多个 JSON 配置时的合并问题。
- 增加网络延时、连接稳定性和远程目录响应监测。
- 改为 Ping 优先、SSH TCP 回退，减少后台 SSH 连接压力。
- 增加瞬时故障复查、慢速分级提醒与状态恢复。
- 增加单实例启动、原子配置保存、配置备份恢复和持久日志。
- 修复失效网络盘检测可能卡住界面的问题。
- 加强字段校验和常见挂载错误说明。
- 增加标准 Windows 安装程序、开始菜单入口、可选桌面快捷方式和卸载入口。
- 安装版把个人配置与程序目录分离，升级或卸载不会打包或删除服务器配置。
- 增加应用图标和 EXE 版本信息。
